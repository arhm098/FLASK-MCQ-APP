how to run app on local host

first change the directory on cmd to inside venv

by running the following commands

cd flask
cd venv

then run the python app in the command prompt

python app.py

then it should give the following output

 * Serving Flask app 'app'
 * Debug mode: off
WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
 * Running on all addresses (0.0.0.0)
 * Running on http://127.0.0.1:5000
 * Running on http://192.168.0.107:5000

 then app should be running on hte following ip on your wifi can have access to the application
